Dossier pour les ressources externes (Lexique).

Placez ici le fichier Lexique 4 (ou Lexique 3) au format CSV ou TSV, par exemple :
  - Lexique404.tsv
  - Lexique400.csv
  - Lexique383.tsv

Téléchargement : https://chrplr.github.io/openlexicon/ ou http://www.lexique.org/

Le script syllabify_lexique.py cherche automatiquement un fichier dont le nom
commence par Lexique4, Lexique400, Lexique383 ou Lexique382, avec extension .tsv ou .csv.

Colonnes attendues (noms reconnus) :
  - orthographe : ortho, orthographe
  - phonétique  : phon, phonétique
  - syllabation : syll, syllab, syllabation, phonsyll
