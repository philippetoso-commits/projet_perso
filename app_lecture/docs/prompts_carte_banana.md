# Prompts Banana — Images pour la carte (gamification)

Copier-coller chaque bloc dans Banana pour générer les assets. Garder le même style en réutilisant la fin du prompt si besoin.

---

## 1. Fond de carte (version détaillée)
```
Detailed children's game map background, top-down view, winding path of small islands on a rich ocean. Ocean with visible waves, foam, and light reflections. Each island has small palm trees or bushes, sandy edges, maybe tiny rocks. Distant horizon with soft mountains or hills. Several fluffy clouds in the sky, sun or sunbeams, one or two small birds. Sky gradient from light blue to soft teal. Decorative details: seashells, sparkles on water, gentle shadows under islands. Dense, illustrated style, pastel but rich colors, many small details, no text. Mobile game background, friendly and magical, 2D illustration, teal blue green and sand palette.
```
**Format conseillé :** 16:9 ou 9:16.  
Si le rendu est encore trop simple, ajoute en fin de prompt :  
`very detailed, rich environment, lots of small elements, storybook illustration style`.

---

## 2. Île / noeud débloqué (avec étoile)
```
Single round island with a golden star on top, soft gradient green and blue, cute game level badge, 2D game UI asset, no text, children app style, pastel colors, isolated on transparent or white background, simple flat illustration.
```
**Format conseillé :** carré ou 1:1 (icône).

---

## 3. Île / noeud verrouillé (cadenas)
```
Round island with small padlock icon on top, gray and light blue muted colors, locked level icon, 2D game UI asset, no text, children app style, same cute rounded shape as unlocked island but desaturated, simple flat illustration.
```
**Format conseillé :** carré ou 1:1 (même taille que débloqué).

---

## 4. Chemin / liaison entre îles
```
Curved wooden bridge or stone path connecting two round islands, soft 2D illustration, pastel colors, game map connector between levels, no text, children game asset, top-down view, simple and clean.
```
**Format conseillé :** rectangle horizontal ou carré (tu pourras le répéter ou le faire pivoter entre les noeuds).

---

## Dossier dans le projet
Enregistrer les images générées dans :
`app_lecture/assets/images/carte/`

Exemples de noms : `carte_fond.png`, `carte_ile_debloquee.png`, `carte_ile_verrouillee.png`, `carte_chemin.png`.
